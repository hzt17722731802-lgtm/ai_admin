## 客服管理接口

### 基础信息
- **接口前缀**: `/service`
- **实体**: `CustomerService`

### 1. 分页查询客服

#### 请求
- **URL**: `/service/page`
- **Method**: `GET`
- **参数**:
  - `pageNum`: 页码（默认：1）
  - `pageSize`: 每页数量（默认：10）
  - `status`: 状态（可选，如 "ONLINE"、"OFFLINE"）

#### 响应
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "records": [
      {
        "id": "cs001",
        "employeeId": "EMP001",
        "employeeName": "张三",
        "status": "ONLINE",
        "currentSessions": 3,
        "totalSessions": 120,
        "avgResponseTime": 2.5,
        "satisfactionRate": 95.5,
        "createTime": "2024-01-15 10:30:00",
        "updateTime": "2024-03-20 15:45:00"
      }
    ],
    "total": 50,
    "size": 10,
    "current": 1,
    "pages": 5
  }
}
```

#### 前端调用示例
```javascript
const fetchCustomerServices = async (pageNum = 1, pageSize = 10, params = {}) => {
  const url = new URL(`${BASE_URL}/service/page`);
  url.searchParams.append('pageNum', pageNum);
  url.searchParams.append('pageSize', pageSize);
  
  if (params.status) url.searchParams.append('status', params.status);
  
  const response = await fetch(url, {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer your_token'
    }
  });
  
  const result = await response.json();
  return result;
};

// 调用示例
fetchCustomerServices(1, 10, { status: 'ONLINE' });
```

## 客服对话接口

### 基础信息
- **接口前缀**: `/service/chat`

### 1. 客服发送对话消息

#### 请求
- **URL**: `/service/chat/message`
- **Method**: `POST`
- **请求参数**:
  - `chatId`: 会话ID（必填）
  - `serviceId`: 客服ID（必填）
  - `content`: 消息内容（必填）

#### 请求示例
```
POST /service/chat/message?chatId=chat001&serviceId=cs001&content=您好，请问有什么可以帮助您的？
```

#### 响应
- **Content-Type**: `text/html;charset=utf-8`
- **响应类型**: 流式响应（Flux）

#### 响应示例
```
发送成功
```

#### 前端调用示例
```javascript
const sendServiceChatMessage = async (chatId, serviceId, content) => {
  const url = new URL(`${BASE_URL}/service/chat/message`);
  url.searchParams.append('chatId', chatId);
  url.searchParams.append('serviceId', serviceId);
  url.searchParams.append('content', content);
  
  const response = await fetch(url, {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer your_token'
    }
  });
  
  const result = await response.text();
  return result;
};

// 调用示例
sendServiceChatMessage('chat001', 'cs001', '您好，请问有什么可以帮助您的？');
```

## 注意事项

1. **会话管理**: 客服接管对话后，所有消息（用户、AI、客服）都使用同一个 chatId，确保对话连贯性

2. **消息存储**: 所有消息统一存储在 chat_history 表中，通过 role 字段区分消息来源（user、assistant、service）

3. **状态说明**:
   - 客服状态：ONLINE（在线）、OFFLINE（离线）
   - 会话状态：ACTIVE（进行中）、COMPLETED（已完成）

4. **权限控制**: 客服接口需要携带有效的认证令牌（Authorization: Bearer your_token）

5. **跨域支持**: 接口支持跨域访问，前端可直接调用

6. **错误处理**: 建议前端对网络错误和业务错误进行统一处理

7. **实时性**: 客服工作台建议定期刷新会话列表和消息，或使用 WebSocket 实现实时更新

8. **性能优化**: 避免频繁调用接口，建议设置合理的轮询间隔（如 5-10 秒）
